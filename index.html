About Editkaro.in · Friendly reels-focused studio

Editkaro.in is a small, independent video editing studio that enjoys working on reels, shorts and social media-first videos. The focus is on:

  • Strong hooks in the first 3 seconds
  • Readable subtitles and clean layouts
  • Simple transitions that fit the beat
  • Delivering files ready to post on social platforms

This website was created as a mini-project and demo portfolio for academic evaluation, but it is structured so it can be shared with real clients in the future.
